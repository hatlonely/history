# markdown


## 语法

    #       h1标题
    ##      h2标题
    ###     h3标题
    >       层次
    >>      层次
    *       无序列表
    -       无序列表
    +       无序列表
    1.      有序列表
    2.      有序列表
    ***     分割线
    -----   分割线
    *强调*
    _强调_
    **加粗**
    ***加粗和强调***
    `代码`
    ``代码段``
    ```代码块```
    4个缩进或者1个制表符表示代码块
    \ 转义

    [link](http://url.net) 链接
    ![img](path) 图片
    [link][id] 链接
    <http://google.com/> 自动链接

    [id]: http://example.com/ 'option title here'
    [id]: http://example.com/ "option title here"
    [id]: http://example.com/ (option title here)
