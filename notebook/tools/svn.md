# svn


## 常用命令

```
svn co
svn st
svn up
svn ci

svn changelist --help
svn changelist <CLNAME> [PATH [PATH ...]]
svn changelist remove <PATH>
svn checkin -m "<message>" --changelist <CLNAME>
svn checkout <url>
svn staus
svn update

svn add
svn rm
svn cp
```
