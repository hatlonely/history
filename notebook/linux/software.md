#linux下必备软件安装

## lrzsz SecureCRT文件传输工具
```bash
$ wget --no-check-certificate https://ohse.de/uwe/releases/lrzsz-0.12.20.tar.gz
$ ./configure
$ make
# make install
```

## python 2.7.8
$ wget https://www.python.org/ftp/python/2.7.8/Python-2.7.8.tgz
$ tar -xzvf Python-2.7.8.tgz
$ cd Python-2.7.8
$ ./configure
$ make
# make install
